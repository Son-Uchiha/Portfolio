[ 04 ] 開発 — Dự án

Dự án nổi bật

NestJS E-Commerce Backend

Hệ thống Backend E-Commerce hoàn chỉnh, thiết kế theo kiến trúc production-ready

NestJS 11 TypeScript 5 Prisma 7 MySQL Redis BullMQ VNPay Zod

Authentication & Authorization

  • Stateless JWT với Redis Blacklist — thu hồi token tức thì
  • Token Rotation cho Refresh Token — chống Replay Attack
  • RBAC đầy đủ (3 tầng Guard): AuthGuard → PermissionsGuard → SuperAdminGuard
  • Forgot Password qua OTP — lưu Redis với TTL auto-expire

Checkout Engine & Thanh toán

  • Checkout trong Serializable Transaction — đảm bảo Data Consistency
  • Tích hợp cổng VNPay: Return URL + IPN Webhook, kiến trúc Idempotent
  • Sử dụng Prisma.Decimal (không dùng Number) — tránh sai số floating-point
  • Xử lý thanh toán thất bại: tự động hoàn kho + hủy đơn trong transaction

Message Queue & Worker Architecture

  • BullMQ (backed by Redis) — xử lý bất đồng bộ, không block API
  • 2 Queue riêng biệt: mail_queue (6 loại email) + order_queue (auto-cancel 15 phút)
  • Worker tách biệt API Server — scale độc lập theo Competing Consumers
  • Worker crash không ảnh hưởng API Server — Fault Isolation

Quản lý Sản phẩm & Admin

  • CRUD sản phẩm: hình ảnh gallery, giá khuyến mãi, tìm kiếm + filter + phân trang
  • Giỏ hàng với Upsert + Atomic Increment — chống Lost Update
  • Admin Dashboard APIs với phân quyền chi tiết
  • State Machine đơn hàng: PENDING → CONFIRMED → SHIPPING → DELIVERED / CANCELLED
